# 🚀 Projeto Backend API

Esta é uma API REST desenvolvida em Node.js para gerenciamento de usuários, categorias e produtos, com sistema de autenticação via JWT.

## 🛠️ Tecnologias Utilizadas

* **Node.js** & **Express** - Framework web.
* **Sequelize** - ORM para manipulação do banco de dados.
* **PostgreSQL** - Banco de dados (hospedado no Supabase).
* **JWT (JSON Web Token)** - Autenticação segura.
* **Bcrypt.js** - Criptografia de senhas.
* **Helmet** - Segurança dos cabeçalhos HTTP.
* **CORS** - Gerenciamento de permissões de acesso.

---

## 📋 Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
* [Node.js](https://nodejs.org/en/)
* Um cliente para testar a API (Postman, Insomnia ou Thunder Client)
